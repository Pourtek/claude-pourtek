8. Keg Forecast

This predicts when active kegs will run out.

Columns:

Beverage
Name of the drink.

% Remaining
How much is left in the keg.

Days Until Empty
Estimated days before it runs out based on recent pour activity.

Example:

Franciscan Cabernet: 3.9% remaining, 3.8 days until empty
That keg is almost gone and should be watched or replaced soon.

This is one of the most useful parts of the dashboard because it prevents the classic hospitality disaster: running out of the thing everyone wants.

9. Slow Movers

This shows old kegs that are not selling quickly.

Columns:

Beverage
The product.

Age Days
How long the keg has been sitting.

Example:

Test Beer: 813 days
That is ancient. At that point, the keg may legally qualify as a historical artifact.

Slow movers help operators decide what to discount, replace, promote, or stop ordering.

This shows everything currently active on tap.

Columns:

Tap

The tap location number.

Beverage

The drink assigned to that tap.

Category

Type of beverage, like Light, IPA, Red Wine, etc.

Style

% Remaining

How much of the keg is left.

Oz Remaining

Remaining ounces in that keg.

This table helps operators answer:

“Which taps are almost empty?”
“Which ones are old?”

18. Restock Forecast by Category

This is probably the most operator-friendly inventory section.

It shows each category and answers:

Projected Empty

How many kegs in that category are expected to run out soon.

Available

How many backup kegs are already available.

Net to Order

How many kegs should be ordered after accounting for available backups.

Example:

Cider
Projected Empty: 2
Available: 0
Net to Order: 2

That means the operator should order 2 cider kegs.

Light
Projected Empty: 4
Available: 8
Net to Order: 0

That means no reorder is needed because there is enough backup inventory.

Simple Translation

The first screen answers:

How much did we sell, what sold best, when were we busiest, and what should we pay attention to?

The second screen answers:

What is on tap, what is left, what is empty, what is old, and what do we need to reorder?

For PourTek sales, the big message is:

Operators can see revenue, pours, customer behavior, inventory risk, keg health, and restocking needs from one dashboard instead of managing it through guesswork, staff memory, and whatever sticky note survived behind the bar.

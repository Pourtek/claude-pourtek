# Claude PourTek

This repository stores the PourTek Claude Skill, Claude Design files, brand assets, product images, knowledge files, templates, examples, and review files.

## Main folders

- pourtek-content-studio: Claude Skill folder
- claude-design: Files to upload into Claude Design
- NEEDS_REVIEWED: Drafts and files that need cleanup before use
- ZIP_UPLOADS: Final ZIP files for Claude Skill uploads

## Upload rules

- Upload pourtek-content-studio.zip to Claude Skills.
- Upload files inside claude-design/upload-to-claude-design to Claude Design.
- Do not upload NEEDS_REVIEWED files into Claude unless reviewed first.
